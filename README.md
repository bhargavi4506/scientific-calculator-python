# Scientific Calculator

A Python-based Scientific Calculator built using Tkinter.

## Features

- Basic Arithmetic Operations
- Scientific Functions
  - Sine
  - Cosine
  - Tangent
  - Logarithm
  - Natural Log
  - Factorial
  - Square Root
  - Exponentials
- Memory Functions (M+, M-, MR, MC)
- History Management
- Export Calculation History
- Dark/Light Theme
- Keyboard Support

## Technologies Used

- Python
- Tkinter
- Math Module

## How to Run

```bash
python calculator.py
```
